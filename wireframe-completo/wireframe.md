# Wireframe do Projeto

O wireframe completo do projeto foi desenvolvido na plataforma Figma e está disponível para visualização através do link abaixo:

**Acesso ao Wireframe:**
https://www.figma.com/design/020llvK3y7SmnyZuXirNHk/Wireframe?node-id=0-1&p=f&t=YUmb5sfl3wEkHR2S-0

O protótipo apresenta toda a estrutura visual da aplicação, incluindo as telas principais, fluxos de navegação, componentes de interface e a organização das funcionalidades propostas para o sistema.

Por meio do wireframe é possível compreender a experiência do usuário planejada para o projeto, visualizar a disposição dos elementos da interface e analisar o fluxo de interação entre as diferentes funcionalidades desenvolvidas.

A ferramenta foi utilizada durante a etapa de planejamento e design da solução, servindo como referência para a implementação do sistema e garantindo maior consistência visual e usabilidade ao produto final.
